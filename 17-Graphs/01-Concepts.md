# 🌐 Graph Concepts

---

# What is a Graph?

A Graph is a collection of:

- Vertices (Nodes)
- Edges (Connections)

---

# Example

```
    A
   / \
  B---C
   \
    D
```

---

# Types of Graphs

- Directed Graph
- Undirected Graph
- Weighted Graph
- Unweighted Graph
- Cyclic Graph
- Acyclic Graph

---

# Terminology

- Vertex
- Edge
- Degree
- Path
- Cycle
- Connected Component

---

# Advantages

- Models relationships naturally
- Flexible structure
- Supports many real-world problems

---

# Complexity

Depends on the graph representation and algorithm.

---

# Interview Tip

Most Graph problems are solved using:

- BFS
- DFS
- Union-Find
