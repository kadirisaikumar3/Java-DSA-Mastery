# Choose the Right Algorithm

| Problem | Algorithm |
|----------|-----------|
| Traversal | BFS / DFS |
| Connected Components | DFS / BFS |
| Cycle Detection | DFS / Union-Find |
| Topological Sort | Kahn / DFS |
| Shortest Path | BFS / Dijkstra |
| MST | Kruskal / Prim |

# Complexity

Most graph algorithms

```
O(V + E)
```
