# 🌳 Minimum Spanning Tree (MST)

---

# What is MST?

A Minimum Spanning Tree is a subset of graph edges that:

- Connects all vertices
- Contains no cycles
- Has the minimum possible total edge weight

---

# Algorithms

- Kruskal's Algorithm
- Prim's Algorithm

---

# Kruskal

Uses:

- Sorting
- Union-Find (DSU)

Complexity

```
O(E log E)
```

---

# Prim

Uses:

- Priority Queue
- Greedy

Complexity

```
O(E log V)
```

---

# Applications

- Road Networks
- Cable Networks
- Network Design
