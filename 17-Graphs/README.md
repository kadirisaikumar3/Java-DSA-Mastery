# 📘 Chapter 17 - Graphs

> "A Graph is a non-linear data structure consisting of vertices (nodes) connected by edges."

---

# 📚 Overview

Graphs are among the most powerful data structures used to model relationships between entities.

They are widely used in social networks, maps, recommendation systems, networking, scheduling, and search engines.
