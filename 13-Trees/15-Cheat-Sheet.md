# 📋 Trees Cheat Sheet

## Traversals

| Traversal | Order |
|-----------|-------|
| Preorder | Root → Left → Right |
| Inorder | Left → Root → Right |
| Postorder | Left → Right → Root |
| Level Order | BFS |

---

## Complexity

| Operation | Time |
|-----------|------|
| DFS | O(n) |
| BFS | O(n) |
| Height | O(n) |
| Diameter | O(n) |

## Pattern Recognition

Need subtree information?

↓

Think DFS

Need level-wise traversal?

↓

Think BFS

Need child results first?

↓

Think Postorder

---

## Golden Rule

Every Tree node is usually visited exactly once in an optimal Tree algorithm.
