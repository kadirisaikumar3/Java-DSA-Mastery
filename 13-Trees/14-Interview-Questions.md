# 🎯 Interview Questions

---

# Beginner

### 1. What is a Tree?

A Tree is a hierarchical, non-linear data structure consisting of nodes connected by edges.

---

### 2. What is the Root Node?

The topmost node of a Tree.

---

### 3. What is a Leaf Node?

A node with no children.

---

### 4. What is the difference between Height and Depth?

- Height: Longest path from a node to a leaf.
- Depth: Distance from the root to a node.

---

### 5. What are the Tree Traversals?

- Preorder (Root → Left → Right)
- Inorder (Left → Root → Right)
- Postorder (Left → Right → Root)
- Level Order (BFS)

---

# Intermediate

### 6. Why is DFS commonly used in Trees?

DFS naturally fits recursive Tree structures and is ideal for subtree-based problems.

---

### 7. When should BFS be used?

When processing nodes level by level or finding the shortest path in an unweighted tree.

---

### 8. What is the Diameter of a Tree?

The number of edges on the longest path between any two nodes.

---

### 9. What is the Lowest Common Ancestor (LCA)?

The deepest node that is an ancestor of both given nodes.

---

### 10. Why is Postorder useful?

Because child information is computed before processing the parent.
