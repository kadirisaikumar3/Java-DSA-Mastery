# 📚 Overview

A Tree is a non-linear data structure used to represent hierarchical relationships. Unlike arrays and linked lists, trees organize data into parent-child relationships.

Trees are the foundation of Binary Search Trees (BST), Heaps, Tries, Segment Trees, and many advanced algorithms.
