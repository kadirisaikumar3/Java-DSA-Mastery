# 🌲 Binary Tree

---

# What is a Binary Tree?

A Binary Tree is a Tree where each node has at most two children.

- Left Child
- Right Child

---

# Node Structure

```java
class TreeNode{

    int val;

    TreeNode left;

    TreeNode right;

    TreeNode(int val){

        this.val = val;

    }

}
```
