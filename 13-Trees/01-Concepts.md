# 🌳 Tree Concepts

---

# What is a Tree?

A Tree is a hierarchical data structure consisting of nodes connected by edges.

Unlike Linked Lists, one node can have multiple children.

---

# Basic Structure

```
          Root
         /    \
      Left    Right
      /  \      \
   Node  Node   Node
```

---

# Terminology

- Root
- Parent
- Child
- Leaf
- Edge
- Height
- Depth
- Subtree
- Ancestor
- Descendant

---

# Binary Tree

A Binary Tree is a Tree in which every node has at most two children.

```
        10
       /  \
      5    15
     / \     \
    2   8     20
```

---

# Advantages

- Hierarchical storage
- Efficient searching
- Recursive structure
- Flexible organization

---

# Disadvantages

- More memory usage
- Pointer management
- Can become unbalanced

---

# Complexity

Depends on the operation and tree type.

Balanced Trees generally perform better than skewed trees.

---

# Interview Tip

Almost every Tree problem is solved using **DFS** or **BFS**.
