# 💼 Google Interview Corner

---

# What Google Evaluates

- Recursive Thinking
- DFS & BFS Knowledge
- Tree Traversals
- Divide and Conquer
- Time Complexity
- Space Complexity

---

# Common Follow-up Questions

- Why DFS instead of BFS?
- Why Postorder?
- Can recursion be avoided?
- Can space be optimized?
- How would this work for an N-ary Tree?

---

# Edge Cases

- Empty Tree
- Single Node
- Skewed Tree
- Balanced Tree
- Duplicate Values

---

# Communication Tips

While solving:

- Draw the tree.
- Explain traversal order.
- Explain recursive return values.
- Mention Time & Space Complexity.

---

# Google Advice

Don't just explain recursion.

Explain:

- What each recursive call returns.
- Why Postorder or Preorder is chosen.
- How subtree results are combined.

This demonstrates mature Tree problem-solving skills.
