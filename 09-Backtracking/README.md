# 📘 Chapter 09 - Backtracking

> "Backtracking systematically explores all possible solutions by making choices, exploring them, and undoing them when necessary."

---

# 📚 Overview

Backtracking is an algorithmic technique that builds solutions incrementally and abandons ("backtracks") partial solutions when they cannot lead to a valid answer.

It is widely used in combinatorial problems where multiple possibilities must be explored efficiently.
