# 🌳 Decision Tree

---

# What is a Decision Tree?

A Decision Tree represents every possible choice made during recursion.

Each node represents a decision.

Each branch represents a possible choice.

---

# Example

Subsets

```
          []
         /  \
       1     X
      / \
     2   X
```

---

# Why Important?

Decision Trees help visualize recursion and understand how Backtracking explores possibilities.

---

# Characteristics

- Root → Initial State
- Branch → Choice
- Leaf → Complete Solution

---

# Applications

- Subsets
- Permutations
- Combination Sum
- N-Queens

---

# Interview Tip

Before coding, draw the Decision Tree for the problem.
