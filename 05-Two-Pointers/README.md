> "Two Pointers is a powerful optimization technique that reduces nested loop solutions into linear-time solutions."

---

# 📚 Overview

The Two Pointer technique uses two indices to traverse an array or string efficiently.

Instead of checking every possible pair, two pointers move according to problem constraints, significantly reducing the time complexity.

It is commonly used in sorted arrays, strings, linked lists, and sliding window problems.
