# Must-Know Data Structures

Choosing the correct data structure is often more important than the algorithm itself.

---

## Linear Data Structures

- Arrays
- Strings
- Linked Lists
- Stack
- Queue

---

## Non-Linear Data Structures

- Trees
- Binary Search Tree
- Heap
- Trie
- Graph

---

## Hashing

Know:

- HashMap
- HashSet

Applications:

- Frequency Counting
- Fast Lookup
- Duplicate Detection

---

## Advanced Data Structures

- Segment Tree
- Fenwick Tree
- Union Find

---

Always choose the data structure before choosing the algorithm.
