# Interview Cheat Sheet

## Time Complexities

| Operation | Complexity |
|-----------|------------|
| Array Access | O(1) |
| Linear Search | O(n) |
| Binary Search | O(log n) |
| HashMap Lookup | O(1) Average |
| Heap Insert | O(log n) |
| DFS | O(V + E) |
| BFS | O(V + E) |
| Merge Sort | O(n log n) |
| Quick Sort (Average) | O(n log n) |
