# 🌳 Introduction to Union Find

---

# What is Union Find?

Union Find, also called **Disjoint Set Union (DSU)**, is a data structure that maintains a collection of disjoint sets.

---

# Primary Operations

- Find
- Union

---

# Why Use Union Find?

- Fast Connectivity Queries
- Efficient Set Merging
- Detect Cycles
- Build Minimum Spanning Trees

---

# Common Applications

- Connected Components
- Kruskal's Algorithm
- Dynamic Connectivity
- Network Problems

---

# Advantages

- Nearly Constant-Time Operations
- Easy to Implement
- Excellent for Graph Problems

---
