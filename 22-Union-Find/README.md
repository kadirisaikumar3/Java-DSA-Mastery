# 📘 Chapter 22 - Union Find (Advanced)

> "Union Find (Disjoint Set Union) is a data structure that efficiently manages dynamic connectivity and disjoint sets."

---

# 📚 Overview

Union Find (DSU) is used to efficiently determine whether two elements belong to the same set and to merge different sets.

It is widely used in graph algorithms and connectivity problems.
