# 📘 Chapter 15 - Heaps

> "A Heap is a Complete Binary Tree that satisfies the Heap Property."

---

# 📚 Overview

A Heap is a specialized tree-based data structure used for efficiently finding the minimum or maximum element.

Heaps are the foundation of Priority Queues and are widely used in scheduling, graph algorithms, and Top-K problems.
