# 🚀 Top Google Heap Problems

---

## 1. Kth Largest Element in an Array

Pattern

```
Min Heap
```

Complexity

```
O(n log k)
```

---

## 2. Top K Frequent Elements

Pattern

```
Frequency Map + Min Heap
```

---

## 3. Merge K Sorted Lists

Pattern

```
Priority Queue
```

---

## 4. Find Median from Data Stream

Pattern

```
Two Heaps
```

---

## 5. K Closest Points to Origin

Pattern

```
Max Heap
```

---

## 6. Task Scheduler

Pattern

```
Greedy + Max Heap
```

---

## 7. Sliding Window Median

Pattern

```
Two Heaps
```

---

## 8. IPO

Pattern

```
Sorting + Max Heap
```

---

# Interview Advice

Focus on identifying whether a problem requires:

- Min Heap
- Max Heap
- Two Heaps
- Priority Queue

Choosing the correct heap pattern is often the key to an optimal solution.
