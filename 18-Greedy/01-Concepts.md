# 💡 Greedy Concepts

---

# What is a Greedy Algorithm?

A Greedy Algorithm makes the best possible local decision at each step.

Once a choice is made, it is never changed.

---

# Key Idea

```
Current Best Choice

↓

Repeat

↓

Optimal Solution
```

---

# Characteristics

- Local Optimization
- No Backtracking
- Fast Execution
- Simple Implementation

---

# Advantages

- Efficient
- Easy to Implement
- Low Memory Usage

---

# Disadvantages

- Does not always produce the optimal solution
- Requires proof of correctness

---

# Complexity

Depends on the problem.

Many Greedy solutions are

```
O(n)

or

O(n log n)
```

---

# Interview Tip

Greedy works only if the problem satisfies:

- Greedy Choice Property
- Optimal Substructure
