# ❌ Brute Force Approach

---

# What is Brute Force?

Try every possible combination.

Choose the best answer after exploring all possibilities.

---

# Example

Jump Game

Try every possible jump recursively.

---

# Complexity

Usually

```
Exponential
```

Examples

```
O(2ⁿ)

O(n!)
```

---

# Drawbacks

- Very Slow
- High Memory Usage
- Not Practical

---

# Interview Tip

Always compare Brute Force with Greedy before choosing your approach.
