# 🎯 Greedy Strategy

---

# Greedy Choice Property

A locally optimal choice should lead to a globally optimal solution.

---

# Optimal Substructure

The optimal solution can be built from optimal solutions of subproblems.

---

# General Steps

1. Sort (if required)
2. Pick the best choice
3. Update the answer
4. Repeat

---

# Example

Activity Selection

Choose the activity that finishes earliest.

---

# Common Patterns

- Sorting + Greedy
- Heap + Greedy
- Interval Scheduling
- Resource Allocation

---

# Interview Tip

Many Greedy problems begin by sorting the input.
