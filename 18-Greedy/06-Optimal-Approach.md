# 🚀 Optimal Approach

---

# General Pattern

1. Sort (if required)
2. Choose the best available option.
3. Never reconsider previous decisions.
4. Continue until finished.

---

# Complexity

Most Greedy algorithms

```
O(n)

or

O(n log n)
```

---

# Examples

- Activity Selection
- Fractional Knapsack
- Jump Game
- Gas Station
- Huffman Coding

---

# Advantages

- Fast
- Simple
- Memory Efficient
