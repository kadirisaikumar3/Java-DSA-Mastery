# 📋 Recursion Cheat Sheet

## Core Components

| Component | Purpose |
|-----------|---------|
| Base Case | Stops recursion |
| Recursive Case | Solves smaller problem |
| Call Stack | Stores recursive calls |

---

## Complexity

| Problem | Time | Space |
|----------|------|-------|
| Factorial | O(n) | O(n) |
| Fibonacci (Naive) | O(2ⁿ) | O(n) |
| Binary Search | O(log n) | O(log n) |

---

## Common Problems

- Factorial
- Fibonacci
- Reverse String
- Sum of Digits
- Binary Search
- Generate Parentheses
- Tower of Hanoi
- Subsets
- Permutations

---

## Pattern Recognition

Need to solve a problem by reducing it into smaller versions?

↓

Think Recursion

Repeated Subproblems?

↓

Think Memoization / Dynamic Programming

---

## Golden Rule

Every recursive function must:

1. Have a Base Case.
2. Reduce the problem size.
3. Eventually terminate.
