# 📘 Chapter 08 - Recursion

> "Recursion is the art of solving a problem by breaking it into smaller instances of the same problem."

---

# 📚 Overview

Recursion is a programming technique where a function calls itself to solve a smaller version of the original problem.

Many advanced algorithms like Tree Traversals, Graph DFS, Backtracking, Divide & Conquer, and Dynamic Programming rely on recursion.

Understanding recursion is essential for becoming a strong problem solver.
