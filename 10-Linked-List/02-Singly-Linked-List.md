# ➡️ Singly Linked List

---

# What is a Singly Linked List?

Each node stores:

- Data
- Next Pointer

---

# Visualization

```
Head

↓

10 → 20 → 30 → NULL
```

---

# Node Structure

```java
class Node{

    int data;

    Node next;

}
```

---

# Operations

- Insert at Beginning
- Insert at End
- Delete Node
- Search
- Traverse
- Reverse

---

# Advantages

- Simple
- Less Memory
- Easy Implementation

---

# Disadvantages

- Cannot Traverse Backward
- Reverse Traversal Difficult

---

# Complexity

| Operation | Time |
|-----------|------|
| Insert Head | O(1) |
| Insert Tail | O(n) |
| Search | O(n) |

---

# Interview Tip

Most coding interview questions use Singly Linked Lists.
