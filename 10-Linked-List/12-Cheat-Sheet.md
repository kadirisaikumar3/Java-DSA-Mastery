## Types

| Type | Features |
|------|----------|
| Singly | Next Pointer |
| Doubly | Previous + Next |

| Operation | Time |
|-----------|------|
| Access | O(n) |
| Search | O(n) |
| Insert Head | O(1) |
| Delete Head | O(1) |
| Reverse | O(n) |
