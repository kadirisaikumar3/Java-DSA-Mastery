# 📘 Chapter 10 - Linked List

> "Linked Lists are dynamic data structures that store elements as nodes connected through pointers."

---

# 📚 Overview

A Linked List is a linear data structure where elements are stored in nodes. Each node contains data and one or more references (links) to other nodes.

Unlike arrays, linked lists do not require contiguous memory allocation, making insertions and deletions efficient.
