# 💼 Google Interview Corner

---

# What Google Evaluates

- Recursive Thinking
- Base Case Design
- Recursive Relation
- Time Complexity
- Space Complexity
- Optimization Skills

---

# Common Follow-up Questions

- Why Recursion?
- Can this be iterative?
- What is the recursion depth?
- Can you optimize using Memoization?
- What is the call stack size?

---

# Edge Cases

- Empty Input
- Single Element
- Base Case
- Large Inputs
- Stack Overflow

---

# Communication Tips

While solving:

- Explain the base case.
- Explain the recursive case.
- Describe the call stack.
- Mention Time & Space Complexity.

---

# Google Advice

Don't just explain what the function does.

Explain:

- Why recursion terminates.
- How each recursive call reduces the problem.
- Why the solution is correct.
