# 💼 Google Interview Corner

---

# What Google Evaluates

- Understanding of BST Property
- Recursive Thinking
- Efficient Search
- Inorder Traversal
- Time Complexity
- Space Complexity

---

# Common Follow-up Questions

- Why use BST instead of Binary Tree?
- Why is Inorder Traversal sorted?
- Can search be iterative?
- How do you validate a BST?
- How do you delete a node with two children?

---

# Edge Cases

- Empty Tree
- Single Node
- Duplicate Values
- Left-Skewed Tree
- Right-Skewed Tree

---

# Communication Tips

While solving:

- Explain the BST property.
- Describe the traversal used.
- Explain recursive return values.
- Mention Time & Space Complexity.

---

# Google Advice

Don't simply write recursive code.

Explain:

- Why you move left or right.
- How the BST property reduces the search space.
- Why the algorithm remains correct after insertion or deletion.

This demonstrates strong problem-solving skills.
