# ❌ Brute Force Approach

---

# What is Brute Force?

A Brute Force solution ignores BST properties and treats the BST like a normal Binary Tree.

---

# Example

Searching for a value

```
Search 8
```

Traverse every node until found.

---

# Complexity

```
O(n)
```

---

# Why is it inefficient?

Even though BST is ordered,

Brute Force does not use the ordering property.

---

# Drawbacks

- Visits unnecessary nodes
- Slower search
- Wastes BST advantages

---

# Interview Tip

Never ignore the BST property.

It helps eliminate half of the search space at every step.
