# 📘 Chapter 14 - Binary Search Tree (BST)

> "A Binary Search Tree (BST) is a Binary Tree where every node satisfies the BST property: Left < Root < Right."

---

# 📚 Overview

A Binary Search Tree (BST) is a special type of Binary Tree that stores elements in sorted order.

This property enables efficient searching, insertion, and deletion operations.

BSTs are widely used in databases, search engines, compilers, and many interview problems.
