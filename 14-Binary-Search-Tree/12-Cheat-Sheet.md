# 📋 BST Cheat Sheet

## BST Rule

```
Left < Root < Right
```

---

## Complexity (Balanced BST)

| Operation | Time |
|-----------|------|
| Search | O(log n) |
| Insert | O(log n) |
| Delete | O(log n) |
| Inorder | O(n) |
