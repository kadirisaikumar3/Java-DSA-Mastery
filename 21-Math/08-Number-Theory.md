# 🔢 Number Theory

---

# What is Number Theory?

Number Theory is the branch of mathematics that deals with properties of integers.

---

# Important Topics

- Divisibility
- Factors
- Multiples
- Prime Numbers
- GCD
- LCM
- Modular Arithmetic

---

# Common Techniques

- Euclidean Algorithm
- Prime Factorization
- Sieve of Eratosthenes
- Binary Exponentiation

---

# Applications

- Cryptography
- Competitive Programming
- Hash Functions
- Number-Based Optimization

---

# Complexity

Most Number Theory algorithms work in

```
O(log n)

or

O(√n)
```
