# ❌ Brute Force Approach

---

# What is Brute Force?

Solve the problem by checking every possible value directly.

---

# Examples

- Checking every divisor
- Testing every number for primality
- Computing powers using repeated multiplication

---

# Complexity

Common complexities

```
O(n)

O(n²)

O(2ⁿ)
```

---

# Drawbacks

- Slow
- High Time Complexity
- Not Suitable for Large Inputs

---

# Interview Tip

Always look for mathematical observations before implementing brute force.
