# 🎲 Combinatorics

---

# What is Combinatorics?

Combinatorics deals with counting and arranging objects.

---

# Permutations

Number of arrangements.

Formula

```
nPr

=

n!

/

(n-r)!
```

---

# Combinations

Number of selections.

Formula

```
nCr

=

n!

/

r!(n-r)!
```

---

# Pascal's Identity

```
nCr

=

(n-1)C(r-1)

+

(n-1)Cr
```

---

# Applications

- Backtracking
- Dynamic Programming
- Probability
- Counting Problems

---

# Interview Tip

Use combinations when order does **not** matter and permutations when order **does** matter.
