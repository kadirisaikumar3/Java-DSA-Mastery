# 🎯 Interview Questions

---

# Beginner

### 1. What is Hashing?

A technique used to map data to a fixed-size value (hash code) for fast lookup.

---

### 2. What is a Hash Function?

A function that converts a key into an index (bucket) in a hash table.

---

### 3. What is HashMap?

A data structure that stores data as **Key → Value** pairs.

---

### 4. What is HashSet?

A collection that stores only unique elements.

---

### 5. Difference between HashMap and HashSet?

| HashMap | HashSet |
|----------|----------|
| Key → Value | Values Only |
| Unique Keys | Unique Values |

---

# Intermediate

### 6. What is Collision?

When two different keys produce the same hash value.

---

### 7. How does Java handle collisions?

Using buckets and treeification (Red-Black Tree) when necessary.

---

### 8. Why is HashMap average O(1)?

Because hash functions distribute keys efficiently across buckets.

---

### 9. What is Frequency Counting?

Counting occurrences using a HashMap.

---

### 10. What is Load Factor?

The ratio of stored entries to bucket capacity before resizing occurs.
