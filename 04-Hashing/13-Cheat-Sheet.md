# 📋 Hashing Cheat Sheet

## HashMap Operations

| Operation | Complexity |
|-----------|------------|
| put() | O(1) Average |
| get() | O(1) Average |
| remove() | O(1) Average |
| containsKey() | O(1) Average |

---

## HashSet Operations

| Operation | Complexity |
|-----------|------------|
| add() | O(1) Average |
| contains() | O(1) Average |
| remove() | O(1) Average |

---

## Common Patterns

| Pattern | Used For |
|----------|----------|
| Frequency Counting | Counting Elements |
| HashMap Lookup | Fast Search |
| HashSet | Duplicate Detection |
| Prefix Sum + HashMap | Subarray Problems |

---

## Top Google Problems

- Two Sum
- Valid Anagram
- Group Anagrams
- Contains Duplicate
- Top K Frequent Elements
- Longest Consecutive Sequence
- Subarray Sum Equals K

## Golden Rule

Whenever you need **fast lookup**, **duplicate detection**, or **frequency counting**, think:

- HashMap
- HashSet
