# 📘 Chapter 04 - Hashing

> "Hashing is one of the most powerful techniques for reducing Time Complexity from O(n²) to O(n)."

---

# 📚 Overview

Hashing is a technique used to store and retrieve data efficiently using a key.

Instead of searching linearly, hashing allows nearly constant-time lookup, insertion, and deletion.

Hashing forms the backbone of many interview problems involving frequency counting, duplicate detection, caching, and fast lookups.
