# 💼 Google Interview Corner

---

# What Google Evaluates

- Range Query Understanding
- Tree Construction
- Recursive Thinking
- Time Complexity Analysis
- Lazy Propagation

---

# Common Follow-up Questions

- Why use Segment Tree over Prefix Sum?
- Why allocate 4 × n memory?
- Why is Build O(n)?
- When should Lazy Propagation be used?
- Difference between Segment Tree and Fenwick Tree?

---

# Edge Cases

- Single Element Array
- Empty Array
- Full Range Query
- Partial Overlap
- No Overlap

---

# Communication Tips

While solving:

- Explain node ranges.
- Describe overlap cases.
- Discuss recursion.
- Mention Time & Space Complexity.

---

# Google Advice

Clearly explain the three query cases:

- No Overlap
- Partial Overlap
- Complete Overlap

Interviewers often focus on this reasoning.
