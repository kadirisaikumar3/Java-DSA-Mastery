# 🌳 Introduction to Segment Tree

---

# What is a Segment Tree?

A Segment Tree is a binary tree used for storing information about intervals (segments) of an array.

It efficiently answers range queries while supporting updates.

---

# Why Segment Tree?

Without Segment Tree:

- Range Query → O(n)

With Segment Tree:

- Range Query → O(log n)
- Point Update → O(log n)

---

# Common Operations

- Build Tree

# Applications

# Advantages

- Fast Queries
- Fast Updates
- Reusable for Different Operations
