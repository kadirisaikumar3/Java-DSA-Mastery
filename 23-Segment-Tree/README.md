# 📘 Chapter 23 - Segment Tree

> "Segment Tree is an advanced tree-based data structure used for efficient range queries and updates."

---

# 📚 Overview

A Segment Tree is a binary tree where each node represents a segment (range) of an array.

It allows efficient:

- Range Sum Query
- Range Minimum Query
- Range Maximum Query
- Point Updates
- Range Updates (using Lazy Propagation)
