# Chapter 24: Fenwick Tree (Binary Indexed Tree)

## 📌 Overview

The **Fenwick Tree**, also known as the **Binary Indexed Tree (BIT)**, is a highly efficient data structure used to perform:

- Prefix Sum Queries
- Point Updates

Both operations can be completed in **O(log n)** time.

Fenwick Trees are simpler to implement than Segment Trees while providing excellent performance for cumulative frequency and range sum problems.
