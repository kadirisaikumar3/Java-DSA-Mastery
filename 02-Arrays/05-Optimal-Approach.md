# 🚀 Optimal Approach

---

# What is an Optimal Approach?

The Optimal Approach is the most efficient known solution for a problem.

It minimizes:

- Time Complexity
- Space Complexity

---

# Characteristics

- Uses the best algorithm
- Handles large inputs efficiently
- Passes interview constraints
