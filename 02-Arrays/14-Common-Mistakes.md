# ❌ Common Mistakes

---

## Mistake 1

Out of Bounds Access

```java
arr[arr.length]
```

Correct

```java
arr[arr.length-1]
```

---

## Mistake 2

Ignoring Edge Cases

- Empty Array
- Single Element
- Duplicate Values
- Negative Numbers

---

## Mistake 3

Using Nested Loops Without Need

Brute Force

```
O(n²)
```

Look for optimization.

---

## Mistake 4

Forgetting Time Complexity

Always analyze your solution.

---

## Mistake 5

Ignoring Space Complexity

Extra arrays increase memory usage.

---

## Mistake 6

Jumping to Code

First understand the problem and discuss your approach.

---

# Interview Advice

Always dry-run your solution with a small example before finalizing it.
