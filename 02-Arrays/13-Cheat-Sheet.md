# 📋 Arrays Cheat Sheet

| Operation | Time Complexity |
|-----------|-----------------|
| Access | O(1) |
| Update | O(1) |
| Search | O(n) |
| Traversal | O(n) |
| Insertion | O(n) |
| Deletion | O(n) |

---

# Common Patterns

| Pattern | Used For |
|----------|----------|
| Traversal | Visiting all elements |
| Prefix Sum | Range Sum Queries |
| Sliding Window | Contiguous Subarrays |
| Two Pointers | Sorted Arrays |
| Binary Search | Sorted Arrays |
| Kadane's Algorithm | Maximum Subarray |
| HashMap | Fast Lookup |
| HashSet | Duplicate Detection |
