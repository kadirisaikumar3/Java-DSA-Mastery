# ❓ Frequently Asked Questions

---

## Q1. Why use a Stack?

To efficiently process data in LIFO order.

---

## Q2. Can Stack be implemented using Arrays?

Yes.

Both Arrays and Linked Lists can implement a Stack.

---

## Q3. What is a Monotonic Stack?

A Stack that maintains sorted order while processing elements.

---

## Q4. Which Stack implementation is preferred in Java?

`ArrayDeque` is generally preferred over `Stack`.

---

## Q5. Why is Stack useful for expression evaluation?

Operators and operands can be processed in the correct order using LIFO.

---

## Q6. Which Stack problems are most important?

- Valid Parentheses
- Next Greater Element
- Daily Temperatures
- Largest Rectangle in Histogram
- Min Stack
- Decode String

---

## Q7. Biggest interview advice?

Don't memorize individual problems.

Learn Stack patterns and recognize when they apply.
