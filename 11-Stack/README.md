# 📘 Chapter 11 - Stack

> "A Stack is a linear data structure that follows the Last In, First Out (LIFO) principle."

---

# 📚 Overview

A Stack stores elements in such a way that the last inserted element is the first one to be removed.

Stacks are widely used in expression evaluation, recursion, browser history, undo operations, and many coding interview problems.
