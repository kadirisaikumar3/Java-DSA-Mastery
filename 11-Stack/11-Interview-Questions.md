# 🎯 Interview Questions

---

# Beginner

### 1. What is a Stack?

A Stack is a linear data structure that follows the **Last In, First Out (LIFO)** principle.

---

### 2. What are the basic Stack operations?

- Push
- Pop
- Peek (Top)
- isEmpty()
- Size

---

### 3. What is the Time Complexity of Push and Pop?

```
Push : O(1)

Pop  : O(1)
```

---

### 4. What is LIFO?

The last element inserted into the stack is the first one removed.

---

### 5. What are the applications of Stack?

- Function Calls
- Undo/Redo
- Browser History
- Expression Evaluation
- DFS
- Syntax Checking

---

# Intermediate

### 6. What is a Monotonic Stack?

A Stack that maintains elements in increasing or decreasing order.

---

### 7. Why use a Monotonic Stack?

To efficiently solve nearest greater/smaller element problems.

---

### 8. What is the difference between Stack and Queue?

| Stack | Queue |
|--------|-------|
| LIFO | FIFO |
| One End | Two Ends |
| Push/Pop at Top | Insert Rear, Remove Front |

---

### 9. Why is ArrayDeque preferred over Stack in Java?

- Faster performance
- No synchronization overhead
- Recommended by Java documentation

---

### 10. What are the most common Stack interview patterns?

- Matching Brackets
- Monotonic Stack
- Expression Evaluation
- Simulation
- DFS
