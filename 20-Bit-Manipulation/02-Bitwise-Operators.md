# ⚙️ Bitwise Operators

---

# AND (&)

Returns 1 only if both bits are 1.

Example

```
5  = 0101

3  = 0011

------------

&  = 0001
```

Result

```
1
```

---

# OR (|)

Returns 1 if either bit is 1.

---

# XOR (^)

Returns 1 if bits are different.

Example

```
5 ^ 3 = 6
```

---

# NOT (~)

Flips every bit.

---
