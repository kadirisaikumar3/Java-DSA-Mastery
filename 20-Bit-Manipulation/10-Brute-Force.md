# ❌ Brute Force Approach

---

# What is Brute Force?

Check every bit or every possible combination directly.

---

# Examples

- Count each bit manually.
- Check all subsets.
- Compare every pair.

---

# Complexity

Often

```
O(n²)

or

O(n × 2ⁿ)
```

---

# Drawbacks

- Slow
- High Memory Usage
- Not Scalable

---

# Interview Tip

Before writing brute force, think whether bitwise operations can simplify the problem.
