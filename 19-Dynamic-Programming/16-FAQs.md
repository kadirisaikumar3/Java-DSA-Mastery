# ❓ Frequently Asked Questions

---

## Q1. What is the difference between Recursion and DP?

Recursion repeatedly solves the same subproblems.

DP stores results and reuses them.

---

## Q2. What is the difference between Memoization and Tabulation?

Memoization is Top-Down.

Tabulation is Bottom-Up.

---

## Q3. Why is DP difficult?

Because identifying the correct state and transition equation requires practice.

---

## Q4. Which DP problems are most important?

- Climbing Stairs
- House Robber
- Coin Change
- LCS
- LIS
- Edit Distance

---

## Q5. Can all recursive problems use DP?

No.

Only problems with overlapping subproblems and optimal substructure.

---

## Q6. Can DP be combined with other techniques?

Yes.

Common combinations include:

- Graph + DP
- Bitmask + DP
- Greedy + DP
- DFS + Memoization

---

## Q7. Biggest interview advice?

Don't memorize DP solutions.

Practice deriving:

- State
- Transition
- Base Cases
