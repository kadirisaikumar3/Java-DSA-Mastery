# 🎯 Interview Questions

---

# Beginner

### 1. What is Dynamic Programming?

Dynamic Programming (DP) is an optimization technique that stores the results of previously solved subproblems to avoid repeated computation.

---

### 2. What are the two properties required for DP?

- Overlapping Subproblems
- Optimal Substructure

---

### 3. What is the difference between Memoization and Tabulation?

- Memoization → Top-Down approach using Recursion + Cache.
- Tabulation → Bottom-Up approach using Iteration + DP Table.

---

### 4. What is a DP State?

A state represents the answer to a smaller subproblem.

---

### 5. What is a State Transition?

A formula used to compute the current state from previously computed states.

---

# Intermediate

### 6. When should DP be used?

When recursion solves the same subproblem multiple times and the problem has optimal substructure.

---

### 7. Why is DP faster than recursion?

Because each subproblem is solved only once.

---

### 8. Can DP be space optimized?

Yes.

Many 1D DP problems only require storing a few previous states.

---

### 9. Which DP pattern is most common?

The Fibonacci pattern is the foundation of many DP problems.

---

### 10. What is the biggest challenge in DP?

Defining the correct state and transition equation.
