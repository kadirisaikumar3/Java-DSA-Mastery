# 🧮 2D Dynamic Programming

---

# What is 2D DP?

2D DP uses a matrix where each state depends on neighboring states.

---

# Common Problems

- Unique Paths
- Edit Distance
- Longest Common Subsequence
- Longest Palindromic Subsequence

---

# General Formula

```
dp[i][j]
```

depends on

```
dp[i-1][j]

dp[i][j-1]

dp[i-1][j-1]
```

---

# Complexity

Time

```
O(n × m)
```

Space

```
O(n × m)
```

---
