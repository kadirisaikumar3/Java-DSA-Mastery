# 🚀 Optimal Approach

---

# DP Workflow

1. Define the State
2. Find the State Transition
3. Define Base Cases
4. Choose Memoization or Tabulation
5. Optimize Space (if possible)

---

# Complexity

Most DP problems

```
O(n)

O(n²)

O(n × m)
```

---

# Advantages

- Eliminates repeated work
- Faster than recursion
- Handles optimization problems efficiently
