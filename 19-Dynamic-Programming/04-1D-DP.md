# 📈 1D Dynamic Programming

---

# What is 1D DP?

1D DP uses a one-dimensional array where each state depends on one or more previous states.

---

# Common Problems

- Fibonacci Number
- Climbing Stairs
- House Robber
- Min Cost Climbing Stairs
- Decode Ways

---

# General Formula

```
dp[i] = function(dp[i-1], dp[i-2], ...)
```

---

# Example

Climbing Stairs

```
dp[1] = 1

dp[2] = 2

dp[i] = dp[i-1] + dp[i-2]
```

---

# Complexity

Time

```
O(n)
```

Space

```
O(n)
```

Can often be optimized to

```
O(1)
```
