# 📘 Chapter 19 - Dynamic Programming

> "Dynamic Programming solves complex problems by breaking them into overlapping subproblems and storing previously computed results."

---

# 📚 Overview

Dynamic Programming (DP) is an optimization technique that avoids repeated calculations by storing intermediate results.

It is widely used in optimization, path finding, string matching, scheduling, and game theory.
