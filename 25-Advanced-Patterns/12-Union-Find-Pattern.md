# Union Find Pattern

Union Find (Disjoint Set Union) efficiently manages disjoint groups while supporting merge operations.

---

## Operations

### Find

Returns the representative (parent) of a set.

### Union

Combines two different sets.

---

## Optimizations

- Path Compression
- Union by Rank
- Union by Size

---

## Applications

- Connected Components
- Cycle Detection
- Kruskal's Algorithm
- Network Connectivity
- Dynamic Graph Problems

---

## Common Problems

- Number of Provinces
- Redundant Connection
- Accounts Merge
- Kruskal MST

---

## Complexity

| Operation | Complexity |
|-----------|------------|
| Find | Nearly O(1) |
| Union | Nearly O(1) |

---

## Key Idea

Union Find is the preferred solution for dynamic connectivity problems.
