# Cheat Sheet

| Pattern | Common Complexity | Typical Problems |
|----------|-------------------|------------------|
| Two Pointers | O(n) | Pair Sum, Palindrome |
| Sliding Window | O(n) | Subarrays, Substrings |
| Fast & Slow | O(n) | Cycle Detection |
| Merge Intervals | O(n log n) | Scheduling |
| Binary Search | O(log n) | Sorted Search |
| Top K Elements | O(n log k) | Heap Problems |
| Monotonic Stack | O(n) | Next Greater Element |
| DFS | O(V+E) | Graph Traversal |
| BFS | O(V+E) | Shortest Path |
| Dynamic Programming | Depends | Optimization Problems |
| Greedy | Usually O(n log n) | Scheduling |
| Union Find | Nearly O(1) | Connectivity |
| Backtracking | Exponential | Permutations |

---

## Pattern Recognition Guide

**Sorted Array**

→ Binary Search / Two Pointers

**Subarray**

→ Sliding Window

**Tree**

→ DFS / BFS

**Graph Connectivity**

→ Union Find

**Intervals**

→ Merge Intervals

**K Largest**

→ Heap

**Optimization**

→ Dynamic Programming

**All Possibilities**

→ Backtracking
