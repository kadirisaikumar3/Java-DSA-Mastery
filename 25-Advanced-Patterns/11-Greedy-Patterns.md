# Greedy Patterns

Greedy Algorithms make the best possible local decision at every step.

If the greedy choice property holds, this produces the global optimum.

---

## When to Use

- Activity Selection
- Huffman Coding
- Minimum Platforms
- Fractional Knapsack
- Jump Game
- Gas Station

---

## Example

Coin Change (Greedy Version)

Always pick the largest denomination first.

```
Coins:

10 5 2 1
```

---

## Common Problems

- Jump Game
- Merge Intervals
- Non-overlapping Intervals
- Gas Station
- Minimum Arrows to Burst Balloons

---

## Complexity

Most greedy algorithms require sorting.

| Time | O(n log n) |

---

## Key Idea

Always verify whether a greedy strategy guarantees an optimal solution before applying it.
