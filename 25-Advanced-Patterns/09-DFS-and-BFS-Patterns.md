# DFS and BFS Patterns

Depth First Search (DFS) and Breadth First Search (BFS) are fundamental graph and tree traversal techniques.

Choosing the correct traversal pattern is essential for solving many interview problems.

---

## DFS

Explores one path completely before backtracking.

Uses:

- Recursion
- Stack

Applications:

- Tree Traversal
- Connected Components
- Backtracking
- Cycle Detection
- Path Finding

---

## BFS

Explores nodes level by level.

Uses:

- Queue

Applications:

- Shortest Path (Unweighted Graph)
- Level Order Traversal
- Minimum Steps Problems
- Multi-source BFS

---

## Common Problems

DFS

- Number of Islands
- Clone Graph
- Path Sum
- Word Search

BFS

- Binary Tree Level Order Traversal
- Rotting Oranges
- Word Ladder
- Open the Lock

---

## Complexity

| Graph Traversal | Complexity |
|-----------------|------------|
| Time | O(V + E) |
| Space | O(V) |

---

## Key Idea

Choose DFS for deep exploration and BFS for shortest-path or level-wise traversal.
