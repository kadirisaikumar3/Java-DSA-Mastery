# 💼 Google Interview Corner

---

# What Google Evaluates

- Pattern Recognition
- Problem Solving
- Communication
- Time Complexity
- Space Complexity
- Code Quality

---

# Common Follow-up Questions

- Why Sliding Window?
- Why not nested loops?
- Can you reduce space complexity?
- Why does the window shrink?
- When should the window expand?

---

# Edge Cases

- Empty Array
- Empty String
- Window Size = 1
- Window Size = Array Length
- Duplicate Characters
- Negative Numbers

---

# Communication Tips

While solving:

- Explain the brute-force solution.
- Explain why Sliding Window is better.
- Describe window expansion.
- Describe window shrinking.
- Mention Time & Space Complexity.

---

# Google Advice

Don't just move the window.

Explain:

- Why it expands.
- Why it shrinks.
- Why every element is processed only once.

That demonstrates true understanding of the algorithm.
