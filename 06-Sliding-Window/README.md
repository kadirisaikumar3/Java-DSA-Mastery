# 📘 Chapter 06 - Sliding Window

> "Sliding Window is one of the most powerful optimization techniques for solving subarray and substring problems in linear time."

---

# 📚 Overview

Sliding Window is an optimization technique that efficiently processes contiguous subarrays or substrings by maintaining a window that expands or shrinks as needed.

Instead of repeatedly recalculating values for every subarray, the window moves across the data, reducing Time Complexity from O(n²) to O(n).
