# 📋 Sliding Window Cheat Sheet

## Window Types

| Type | Use Case |
|------|----------|
| Fixed Size | Maximum Sum Subarray |
| Variable Size | Longest Substring |

## Complexity

| Operation | Complexity |
|-----------|------------|
| Traversal | O(n) |
| Extra Space | O(1) to O(n) |

## Common Problems

- Maximum Sum Subarray of Size K

## Pattern Recognition

Given Window Size?

↓

Fixed Size Window

Need Longest / Shortest Window?

↓

Variable Size Window

Need Character Frequency?

↓

Sliding Window + HashMap

---

## Golden Rule

Every element should enter and leave the window at most once.
