# 📘 Chapter 12 - Queue

> "A Queue is a linear data structure that follows the First In, First Out (FIFO) principle."

---

# 📚 Overview

A Queue stores elements in the order they arrive. The first element inserted is the first one removed.

Queues are widely used in scheduling, buffering, breadth-first search (BFS), operating systems, networking, and many coding interview problems.
