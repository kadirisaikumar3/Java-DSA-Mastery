# 📋 Queue Cheat Sheet

## Queue Principle

```
First In

↓

First Out (FIFO)
```

| Operation | Time |
|-----------|------|
| Enqueue | O(1) |
| Dequeue | O(1) |
| Front | O(1) |
| Rear | O(1) |

Need FIFO processing?

↓

Think Queue

Need BFS?

↓

↓

Think Deque

Need priority-based removal?

↓

## Golden Rule

Each element should enter and leave the queue only once in optimal queue algorithms.
