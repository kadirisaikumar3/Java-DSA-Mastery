# 📋 Binary Search Cheat Sheet

## Binary Search Variants

| Variant | Use Case |
|----------|----------|
| Classic | Search Element |
| Lower Bound | First ≥ Target |
| Upper Bound | First > Target |
| Boundary | First / Last Occurrence |
| Search on Answer | Optimization Problems |

---

## Complexity

| Operation | Complexity |
|-----------|------------|
| Search | O(log n) |
| Extra Space | O(1) |

## Pattern Recognition

Sorted Array?

↓

Binary Search

Need First/Last Position?

↓

Boundary Search

Need Minimum / Maximum Valid Answer?

↓

Search on Answer

---

## Golden Rule

Each iteration removes **half** of the remaining search space.
