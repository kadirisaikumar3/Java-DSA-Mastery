# 💼 Google Interview Corner

---

# What Google Evaluates

Interviewers focus on:

- Correctness
- Time Complexity
- Space Complexity
- Problem Solving
- Communication
- Clean Code

---

# Typical Interview Flow

1. Understand the problem.

2. Discuss brute-force.

3. Optimize.

4. Analyze complexity.

5. Test edge cases.

---

# Common Follow-up Questions

- Can you optimize your solution?
- Can you reduce memory usage?
- Can this work for Unicode characters?
- What if the string is empty?
- What if there are duplicate characters?

---

# Communication Tips

While solving:

- Explain your approach.
- Justify your optimization.
- Mention Time Complexity.
- Mention Space Complexity.

---

# Google Advice

Never jump directly into coding.

Spend a few minutes discussing the approach before writing code.
