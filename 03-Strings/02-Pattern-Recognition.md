# 🧩 Pattern Recognition

---

# Why Learn Patterns?

Most String interview problems follow a small number of recurring patterns.

Recognizing the correct pattern leads to faster and cleaner solutions.

---

# Pattern 1

Character Frequency

Problems

- Valid Anagram
- Ransom Note

Technique

```
HashMap

or

Frequency Array
```

# Interview Tip

Before coding ask:

- Is frequency counting useful?

- Can Sliding Window solve it?

- Can Two Pointers reduce complexity?

- Can StringBuilder improve performance?
