# 📋 Strings Cheat Sheet

| Operation | Complexity |
|-----------|------------|
| length() | O(1) |
| charAt() | O(1) |
| equals() | O(n) |
| substring() | O(n) |
| contains() | O(n) |
| indexOf() | O(n) |
| replace() | O(n) |
| split() | O(n) |
| reverse() | O(n) |
