# 📚 Overview

A String is a sequence of characters used to represent text.

In Java, Strings are immutable objects, making them safe, efficient, and widely used throughout software development.

From validating passwords to searching text and processing user input, Strings appear in almost every application.

This chapter covers String fundamentals, interview patterns, Java methods, optimization techniques, and Google interview questions.
